# 🚀 Company Registration System

Sistema administrativo para cadastro e gerenciamento de empresas, desenvolvido com foco em boas práticas de front-end, validação de formulários e organização de código.

---

## 🧠 Tecnologias utilizadas

* ⚛️ React
* 🔷 TypeScript
* ⚡ Vite
* 🎨 Tailwind CSS
* 📋 React Hook Form
* ✅ Zod
* 🧩 Context API

---

## 📋 Funcionalidades

* Cadastro completo de empresas
* Cadastro de clientes e fornecedores
* Validação de formulário com Zod
* Máscaras para CPF/CNPJ e telefone
* Preenchimento automático de endereço via CEP
* Upload de logo
* Listagem de dados (empresas, clientes e fornecedores)
* Estrutura reutilizável com componentização
* Gerenciamento de estado com Context API

---

## 🧩 Arquitetura do projeto

O projeto foi estruturado seguindo boas práticas de organização:

* Componentes reutilizáveis (inputs, formulários, UI)
* Separação de responsabilidades
* Contextos para gerenciamento de dados
* Tipagem com TypeScript
* Estrutura escalável para crescimento (mini ERP)

## 🎯 Objetivo

Este projeto foi inspirado em um sistema real utilizado em ambiente profissional, sendo adaptado com dados fictícios para fins de estudo e portfólio.

O foco principal foi praticar:

* Criação de formulários complexos
* Validação robusta de dados
* Organização de código em projetos reais
* Simulação de um sistema administrativo (ERP)
